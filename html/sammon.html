Introduction

Multidimensional Scaling (MDS) is a method of embedding the distance information of a multi-variate data-set, in a reduced dimension \(L\), by seeking a set of vectors \( \{\mathbf{\hat{x}}\} ^n_{i=1} \in \mathbb{R} ^L \) that reproduce these distances. The Stress measure \(S\) is represented by,

$$ S_{sam} = \frac{1}{\sum _{i < j} d ^2 _{ij}} \sum _{ i < j} \frac{(d_{ij} - \hat{d} _{ij}) ^2}{d _{ij}}$$

which is again minimised, using a suitable gradient descent technique.
